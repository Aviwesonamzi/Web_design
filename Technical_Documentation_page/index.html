Introduction

JavaScript is a versatile programming language that is used to create dynamic and interactive content on the web. It is an essential part of modern web development.

JavaScript was initially created to "make web pages alive". The programs in this language are called scripts. They can be written right in a web page's HTML and run automatically as the page loads.

Scripts are provided and executed as plain text. They don't need special preparation or compilation to run.

In this documentation, we will cover the basics of JavaScript, its syntax, and its key features.

Syntax and Operators

JavaScript syntax is the set of rules that define a correctly structured JavaScript program. It includes:

  • Variables and Constants
  • Operators
  • Control Structures
  • Functions
  • Comments

JavaScript uses the following types of operators:

  • Arithmetic Operators: +, -, *, /, %
  • Comparison Operators: ==, !=, ===, !==, >, <, >=, <=
  • Logical Operators: &&, ||, !

Here is an example of an arithmetic operation:

let sum = 10 + 5; // 15
Variables

Variables in JavaScript are used to store data values. You can declare a variable using the var, let, or const keywords.

For example:

let message = "Hello, World!";

The let keyword is used to declare variables that can be reassigned. The const keyword is used to declare variables that are read-only.

Here is an example:

const pi = 3.14;
Functions

Functions are reusable blocks of code that perform a specific task. They are defined using the function keyword.

For example:

function greet(name) {
  return "Hello, " + name + "!";
}

Functions can take parameters and return values.

Here is how you can call the greet function:

let message = greet("Alice"); // "Hello, Alice!"
Objects and Arrays

Objects and Arrays are the two most important data types in JavaScript. Objects are used to store collections of data and more complex entities.

For example:

let person = {
  firstName: "John",
  lastName: "Doe"
};

Arrays are used to store multiple values in a single variable.

For example:

let fruits = ["Apple", "Banana", "Cherry"];
